#CURRENT WEATHER APP

THis is a Weather App allows the user to search for the current weather for a city whenever you are,for any
This is the simple weather app made by using HTML5, CSS3, Javascript and using the API from OpenWeatherMap API but
please create your own API Key from OpenWeatherMap.
this weather app find  
*max/min temperature.
*wind speed.
*humidity.
*pressure of the wind.
